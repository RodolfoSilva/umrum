Getting started

UMRUM has a few easy ways to quickly get started. Read through to get it up and running in minutes.

Requirements / Dependencies

  • NodeJS >= 0.10
  • Redis (you need to run a local instance of Redis before running the app)
  • MongoDB (it's necessary to define env variable MONGO_URI before running the app or tests - alternatively you can run a local MongoDB)
  • Git

Installation

  $ git clone https://github.com/umrum/umrum.git
  $ cd umrum
  $ npm install

Running

  $ grunt server

UMRUM will be available at http://localhost:8000.

You can also configure Grunt to run before any commit (pre-commit hook):

  $ echo "grunt || exit 1" > .git/hooks/pre-commit
  $ chmod +x .git/hooks/pre-commit

Testing

  $ grunt unittest
